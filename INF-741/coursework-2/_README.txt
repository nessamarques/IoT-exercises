This instruction was done to run on Linux.

Before you run this project you need:
-Set your ARDUINO_PATH
-Put the button of Player 1 on pin 2
-Put the button of Player 2 on pin 3
-Put the buzzer on pin 8.
-Connect your Arduino Duo on USB

Then, run "compile.bat main" on terminal, the script will compile the project and send it to your Arduino.

To see the prints you need to open Monitor Serial of Arduino IDE or use "screen" on Windows terminal.
