This instruction was done to run on Windows.

Before you run this project you need:
-Put the Encoder A on pin 2.
-Put the Encoder B on pin 3.
-Put the Servo on pin 9.
-Connect your Arduino Duo on USB

To see the prints you need to open Monitor Serial of Arduino IDE.
