Please follow these steps:

1- Connect the Arduino to the serial port of your computer.
2- In the arduino:
	- Connect the button to pin 2
	- Connect the buzzer to pin 3.
3- Extract the contents of the file "coursework-3.zip"
4- Arduino IDE:
	- Upload the file "coursework-3.ino"
	- View the software operation in the "Serial monitor".

Note: These instructions works in Windows OS.
